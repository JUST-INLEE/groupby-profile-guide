📋 경력으로 작성 — 실무 경력이 있는 경우
회사명·포지션·기간을 채우고 추가 정보 → 상세 설명(업무 및 성과)을 반드시 작성하세요. 경력기술서 내용을 여기에 옮겨 적는다고 생각하면 돼요.
💡 관련 활동으로 작성 — 신입/경력 없는 경우
사이드 프로젝트, 교육 수료, 공모전, 대외활동 등 직무와 관련된 모든 경험을 여기에 작성하세요.
⭐ STAR 기법으로 작성하면 훨씬 강력해요
Situation(상황) → Task(과제) → Action(내 행동) → Result(결과)
❌ "마케팅 업무 수행"
✅ "퍼포먼스 캠페인 ROAS 목표 미달 문제를 해결하기 위해 타겟 세그먼트를 재설정하고 소재 A/B 테스트 10회 진행 → ROAS 180% 달성"
❌ 이렇게 쓰면 검수에서 걸려요
교육명 + 기술 스택만 나열 → "OO아카데미 / Figma, Python 학습"
✅ 프로젝트명 + 역할 + 결과물로 바꿔야 해요
